# ☕ Dashboard Financeiro - Serenatto Café

## 📌 Visão Geral
Dashboard analítico desenvolvido em Excel para monitoramento da performance de vendas e saúde financeira da cafeteria. O painel consolida os dados de transações para apresentar indicadores de rentabilidade e fluxo de caixa.

## 📊 Indicadores e Análises (KPIs)
O dashboard apresenta os seguintes dados estratégicos:

### 1. Indicadores Financeiros (Cards Principais)
- **Faturamento Total:** Receita bruta acumulada.
- **Despesas:** Custos operacionais registrados.
- **Margem de Lucro:** Percentual de retorno sobre o investimento.

### 2. Análises de Vendas
- **Top 5 Produtos Mais Vendidos:** Ranking por volume de saída.
- **Ranking de Faturamento:** Classificação dos produtos que mais geram receita (ex: 1º Café, 2º Chocolate Quente).

### 3. Fluxo de Caixa
- **Faturamento x Despesas:** Comparativo visual para identificar a margem líquida.
- **Entrada x Saída Mensal e Anual:** Evolução temporal das movimentações financeiras.

## 🛠️ Estrutura do Projeto
- **Base de Dados:** Controle de Entradas (Compras), Saídas (Vendas) e Cadastro de Produtos/Fornecedores.
- **Processamento:** Cálculos de totais e consolidação via Tabelas Dinâmicas.
- **Visualização:** Gráficos de barras e cartões de KPIs interativos.
